Your AI Does the Work.
You See the Results.

A fully automated system that tracks CPM advertising prices, emails you a weekly report, and keeps score of every hour of work AI saves you — all without you lifting a finger.

The plain-English version: Imagine you had an assistant who, every Monday morning, checked prices at dozens of advertising networks, wrote you a summary email, and also kept a running log of everything else you asked AI to do — with a dollar value attached to each task. That's exactly what this system does. It runs in the cloud, 24/7, on any computer, with zero manual steps.

What Problem Does This Solve?

Two problems, one system.

🔍 Problem 1 — CPM Price Tracking

CPM (Cost Per Thousand impressions) is what advertisers pay publishers for ad space. These prices change constantly, and checking them manually means visiting dozens of websites, copying numbers into a spreadsheet, and sending yourself an email. It's tedious, it's slow, and it's easy to miss updates.


The solution: The system does all of this automatically every Monday at 8am. It searches the web, extracts the numbers, stores them in a cloud database, and emails you a formatted report — with no action required on your end.

📊 Problem 2 — Proving AI's Value

You're using AI tools (Claude and others) every day to write code, analyze data, and build things. But how much time and money is AI actually saving you? Without tracking, it's just a gut feeling.


The solution: Every time you work with Claude in a Cowork session, it automatically logs what was built, estimates how long a human would have taken, and stores that dollar value in the database. You get a real running total — no manual entry required.

What Was Built

Six major components, built in order, each building on the last.

1

Automated CPM Report

A serverless program that wakes up every Monday at 8am, searches the web for CPM prices across multiple advertising channels (video, display, CTV, social), saves every number to a cloud database, and emails you a formatted HTML report with trends. 210+ historical data points going back years. Runs completely in the cloud — no computer needs to be on.

✓ Live
2

Cloud Database (Neon Postgres)

A cloud database that stores every CPM data point ever collected, and also logs every AI interaction. Think of it as a spreadsheet that lives in the cloud, never gets lost, and can be queried from anywhere. 210+ CPM rows migrated from local files; now the single source of truth for all data.

✓ Live
3

AI Interaction Tracker

A logging system that records every significant AI session — what was built, what type of task it was (code, analysis, research, document), how many hours a skilled human would have taken, and the dollar value at consulting rates. Currently showing 11 logged interactions worth $4,261 in estimated value.

✓ Live
4

Real-Time Dashboard

A live HTML page that connects directly to the database and shows you up-to-the-minute stats: how many AI interactions happened, total value generated, ROI multiple, first-pass accuracy, and recent CPM run history. Open it any time and click Refresh — it fetches fresh data instantly.

✓ Live
5

Automatic Session Logging via GitHub

The most powerful piece: a system that logs Claude Cowork sessions to the database automatically, on any computer, with zero manual steps. When a session ends, Claude writes a summary JSON file, commits it to GitHub, and a GitHub Action (a cloud job) picks it up, posts it to the database, and moves the file to an archive folder. Works on any Mac, any machine — completely computer-independent.

✓ Live — validated July 2026
6

18-Skill Reference System

A complete library of 18 reusable "skills" — step-by-step instruction sets that tell Claude exactly how to do recurring tasks (deploy to Vercel, schedule a Mac job, write a Word doc, etc.). Each skill is documented and packaged. An interactive HTML guide makes them searchable and browsable.

✓ Documented & Packaged

How It Works — Plain English

No technical knowledge required to understand this.

The CPM Report Flow (runs every Monday 8am ET)

⏰
Monday 8am
A timer in the cloud fires automatically. No computer needs to be on.
→
🔍
Web Search
The system queries the Brave Search API for current CPM prices across 7+ channels.
→
🗄️
Save to Database
New data points are written to Neon Postgres. Old data is never deleted.
→
📧
Email Report
A formatted HTML email lands in your inbox with trends, insights, and channel breakdown.


The AI Session Logging Flow (happens after every Cowork session)

💬
Cowork Session
You work with Claude. Claude tracks what was built, start time, and end time automatically.
→
📝
Session File
Claude writes a structured log file with task descriptions and value estimates to a folder in the repo.
→
🚀
GitHub Push
You push to GitHub. A cloud job (GitHub Actions) automatically detects the new log file.
→
✅
Posted to Database
The job sends the data to the database. File moves to "posted" folder. Done.

What Happens Automatically vs. What Requires You

Most of the system runs itself.

✅ Fully Automatic (Zero Action Required)

  • Weekly CPM report email every Monday
  • Web price scraping across 7+ channels
  • Database writes for every new data point
  • Session logs posted to database on git push
  • Session file moved to archive after posting
  • Cloud database backup (Neon handles this)
  • Mac backup runner (launchd, also fires Monday 8am)

👤 Requires One Action From You

  • git push — after a Cowork session, to trigger the session logger
  • Vercel redeploy — needed after code changes to the report (rare)
  • Token rotation — GitHub token should be rotated periodically for security
  • Open the dashboard — it doesn't email you; you open it when you want to see metrics
